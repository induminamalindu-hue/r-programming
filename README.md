# 📊 R Programming Fundamentals & Data Analysis

Welcome to the **R Programming** repository! This repository is designed to store, manage, and showcase various R scripts, data analysis workflows, and statistical computing projects. Whether you are learning R from scratch or building custom data models, this project serves as a structured guide for R development.

---

## 🚀 Project Overview

R is a powerful environment for statistical computing, data visualization, and predictive modeling. This repository contains structured code snippets, basic concepts, and practical implementations covering:
* **Core Syntax & Variables** - Understanding R data structures.
* **Data Types** - Working with Vectors, Matrices, Lists, Data Frames, and Factors.
* **Control Flows** - Utilizing conditional statements (`if-else`) and loops (`for`, `while`).
* **Data Manipulation** - Importing, exporting, and cleaning datasets.
* **Data Visualization** - Generating charts, histograms, and scatter plots using standard R graphics or packages like `ggplot2`.

---

## 🛠️ Getting Started & Prerequisites

To run the scripts in this repository locally, you need to install R and optionally RStudio (recommended).

### 1. Installation
* Download and install **R**: [CRAN Project](https://cran.r-project.org/)
* Download and install **RStudio IDE**: [Posit / RStudio](https://posit.co/download/rstudio-desktop/)

### 2. Clone the Repository
Run the following command in your terminal or Git Bash to clone this project:
```bash
git clone [https://github.com/induminamalindu-hue/r-programming.git](https://github.com/induminamalindu-hue/r-programming.git)
cd r-programming
```

### 💻 How to Use the Code

1.Open RStudio.

2.Go to File > Open File... and select any .R script from the repository.

2.Highlight the lines of code you want to test and press Ctrl + Enter (Windows) or Cmd + Enter (Mac) to run them in the console.

Example Code Snippet:
```
# Creating a simple numeric vector and calculating the mean
my_data <- c(12, 15, 18, 22, 25, 30)
mean_value <- mean(my_data)
print(paste("The mean value is:", mean_value))
```
